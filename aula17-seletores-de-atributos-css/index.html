titulo 1

Combinador Irmão Adjacente O combinador irmão adjacente + seleciona o elemento que vem imediatamente depois do primeiro elemento. 

                h2 + p {
                  color: green;
                }


Combinador Irmão Geral O combinador irmão geral ~ é mais abrangente, selecionando todos os elementos que seguem o primeiro. 

                h2 ~ p {
                    color: green;
                    }

titulo 2

Combinador de Filho O combinador de filho > seleciona apenas os elementos que são filhos diretos do primeiro elemento. 

                div > p {
                  color: blue;
                }

Combinador de Descendente O combinador de descendente (espaço) é o mais generoso, selecionando todos os elementos que são descendentes do primeiro, sejam filhos diretos ou não. 

            div p {
            color: blue;
            }
